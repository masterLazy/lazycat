引入：匀加速直线运动

假设有一辆汽车，原先是静止的。现在它启动，并以每 1m/s2 的加速度加速，60s 后，它驶过的路程是多少？

问题中汽车不做匀速直线运动，而是做匀加速直线运动，即它的加速过程中速度保持不变。如果你学过匀加速直线运动，你或许知道在匀加速直线运动中求物体移动的路程的公式：s=12at2。因此，它驶过的路程 s2=12×1m/s2×60s=60m。

你或许会疑惑，“s=12at2”这个公式是怎么来的？为什么要乘以 12？平方项又从何而来？实际上，这是一个值得探讨的问题，并且是积分学所研究的一个简单的问题。

让我在这里演示一下它简要的推导过程。

设加速度为 a ，则记任意时刻 t 时的速度

v(t)=at.

所以任意时刻 t 移动的路程

s(t)=∫vdt=12at2.

其中的 ∫ 是积分符号。欲知详情，请学习微积分（硬核引入）。

一、微分

微分与积分互为逆运算，并且你应当先学习微分。理解了微分，才好用逆运算的思想来学习积分。

切线、斜率与导数

先来看函数 f(x)=x2 的图象：

图中的抛物线（开口向上）即函数 f(x)=x2 的图象，而直线是函数 f 在 x0 处的切线，或者说是过点 P1 的切线。这里可以理解为：切线是一条与函数的某一点最吻合的直线。如果切线是水平的，则说明这一点是函数的一个拐点。在这个例子中，函数 f 有且仅有一个拐点，它在 x=0 处。

假如该切线的方程为 y=kx+b ，则 k 叫做它的斜率。若 k>0，则切线向上；若 k<0，则切线向下。这里可以理解为：斜率就是瞬时增长率。

更准确地说，k 是函数 f(x) 在 x=x0 处的斜率。在高中，我们将它记作 f′(x0)。函数 f′ 就叫做 f 的导函数，简称导数。导数用于求函数在某点的斜率。

求导与极限

我们已经弄懂了什么导数，那么我们如何求它呢？

让我们从切线入手，试着求出它的斜率的表达式，即函数的导数。但是，我们知道，两点才能确定一条直线，而我们要求的是”某点”的切线（或许你已经发现这一点了），我们该如何处理呢？让我们再来看看下面这幅图：

在图中，我们在函数 f 上取了两个点，它们分别是：P1(x0,f(x0)) 、P2(x0+dx,f(x0+dx))。从 P1 到 P2，横坐标 x 增长了 dx（至于什么要起这个名字，我们后面再讲），纵坐标 y 增长了 f(x0+dx)−f(x0)（我们不妨把它记作 df）。则，过这两点的直线的斜率为 dfdx。这里可以理解为：x 增长了 dx，f 跟着增长了 df，增长率就是 dfdx。

前文说到，导数用于求函数在某点的斜率，可以记作 f′。到了这里，我们便能得到

f′=dfdx=ddxf.

在采用最后一种书写方式时，为了避免歧义，若函数 f 包含多项，需加上一层括号。

到这里，为我们还是在求过两点的切线的斜率。但是，如果让 P2 不断靠近 P1，这一条切线会不断趋近于我们梦寐以求的过 P1 的切线。

我们将这条切线的斜率的表达式展开：

dfdx=f(x0+dx)−f(x0)dx.

在这个例子中，我们代入 f(x)=x2，得

dfdx=f(x0+dx)−f(x0)dx=(x0+dx)2−x02dx.

化简，得

dfdx=2x0+dx.

上文说道，如果让 P2 不断靠近 P1，这一条切线会不断趋近于过 P1 的切线。所以，当 dx 越来越趋近于 0 时，所得的切线会越来越趋近过 P1 的切线。观察发现，当 dx 趋近于 0，时，表达式的值会越来越趋近于 2x。

所以，函数 f 的导数为

dfdx=2x.

式中的 x0 表示，当 x=x0 时，此处的斜率为 2x0。

我们在 x、f 前加的前缀“d”，表示这个量是非常非常微小的。这样一来，求得的切线与实际切线的误差也会越来越小。这种思想（即上文加粗的片段）叫做极限（Limit，数学上记作 lim）。

用极限的形式来写我们求函数 df 的导数的过程，即

dfdx=limΔx→0f(x0+Δx)−f(x0)Δx.

式中，limΔx→0 意思是整个式子表示的是 f(x0+Δx)−f(x0)Δx 中 Δx 趋近于 0 时的值。

式中，我们令 Δx 趋于 0，使它越来越微小。这种思想，也叫做微分。在微分学中，d 是一个特殊的符号，表示一个微小的量。在适当的时候，我们将其当作 0 处理。这里可以理解为：d(...) 就是一个 Δ(...)，并且还随身带着一个 limΔ(...)→0。

为了避免歧义，若 (...) 包含多项，需加上一层括号，如：d(x2)。

关于极限的“矛盾”

在极限思想，尤其是微分思想中，有一点需要尤其注意：dx≠0。因而，它可以作分母，并且是有意义的（因为它趋近于零 0，但始终不等于 0，始终有一个 ≠0 的值）。但是，我们能在计算的最后将它视为 0 来处理，因为它是一个非常小的值。

这个矛盾比较令人困惑，为什么这个 dx 一会儿不是 0，一会儿又是 0？让我们回到 dfdx=(x0+dx)2−x02dx 的时候。在 x=2 处，有

dfdx=(2+dx)2−4dx.

按照上文推导出来的结果，dfdx=2x=2×2=4。

求导公式与法则

下面列出了 7 个基本初等函数的导数公式（a 为常数）：

(1)ddxa=0.
(2)ddx(ax)=a.
ddxxa=axa−1.
(4)ddxsin⁡x=cos⁡x.
(5)ddxcos⁡x=−sin⁡x.
(6)ddxax=axln⁡a.
(7)ddxloga⁡x=1xln⁡a.

下面是求导的加法法则：

(8)ddx(f(x)±g(x))=dfdx±dgdx.

下面是求导的乘积法则（口诀：右乘左导，左乘右导）：

(9)ddx(f(x)g(x))=g(x)dfdx+f(x)dgdx.

下面是求导的链式法则：

(10)ddxf(g(x))=dfdgdgdx.
